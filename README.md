# Caffeine Flow

Prérequis
- Node.js 
- npm
- MongoDB (local ou distant)


git clone https://github.com/d-hout/pII-caf-ine-flow.git
cd pII-caf-ine-flow
```

S'assurer que MongoDB est démarré et écoute sur le port 27017 

Lancer le back


cd back

npm run dev  


Lancer le front

cd ../frontend
npm install
npm run dev
